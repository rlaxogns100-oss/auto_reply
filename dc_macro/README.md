# 디시인사이드 자동 댓글 매크로

한석원 갤러리의 모든 최신 글에 자동으로 댓글을 다는 프로그램입니다.

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. Chrome 브라우저가 설치되어 있어야 합니다.

## 사용 방법

1. `config.py` 파일에서 설정을 조정합니다:
   - `COMMENT_TEXT`: 댓글 내용
   - `CHECK_INTERVAL`: 새 글 확인 주기 (초)
   - `HEADLESS`: 브라우저 화면 표시 여부

2. 프로그램 실행:
```bash
python main.py
```

3. 종료하려면 `Ctrl+C`를 누릅니다.

## 주의사항

- 디시인사이드의 이용약관을 준수해주세요.
- 과도한 댓글 작성은 계정 제재의 원인이 될 수 있습니다.
- 적절한 `CHECK_INTERVAL`을 설정하여 서버에 부담을 주지 않도록 합니다.
- 이미 처리한 게시글은 `processed_posts.json` 파일에 저장되어 중복 댓글을 방지합니다.

## 작동 원리

1. 주기적으로 갤러리의 최신 글 목록을 확인합니다.
2. 새로운 글을 발견하면 해당 글에 접속합니다.
3. 설정된 댓글을 자동으로 작성합니다.
4. 처리한 글은 기록하여 중복 처리를 방지합니다.
